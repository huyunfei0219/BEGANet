# BEGANet: Synergistic Boundary Enhancement and Adaptive Gating for Multi-Modal Salient Object Detection

Official PyTorch implementation of **BEGANet** for RGB-D and RGB-T Salient Object Detection (SOD).

---

## 📌 Overview

This repository provides the implementation of **BEGANet**, a unified framework for multi-modal salient object detection. The proposed method integrates:

* **Contextual Boundary Enhancement Module (CBEM)** for explicit boundary modeling
* **Enhanced Dynamic Gate (EDG)** for adaptive multi-modal fusion
* **Learnable Upsampling Decoder** for fine-grained spatial reconstruction

BEGANet achieves state-of-the-art performance on multiple RGB-D and RGB-T benchmark datasets.

---

## 🔥 Highlights

* ✅ Boundary-aware feature learning
* ✅ Adaptive RGB-Depth/Thermal fusion
* ✅ Lightweight and efficient architecture
* ✅ Strong generalization across datasets

## 🧩 Key Components

### CBEM

Enhances boundary awareness and improves structural consistency.

### EDG

Adaptively balances multi-modal contributions.

### Learnable Upsampling

Recovers fine spatial details beyond fixed interpolation.
